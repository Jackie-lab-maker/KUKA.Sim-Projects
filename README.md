# KUKA.Sim-Projects

> Automation and manufacturing simulation projects conducted using KUKA.Sim, covering robot programming, fixture design, and process simulation from internship work at OPmobility and personal engineering projects.
>
> ---
>
> ## Overview
>
> This repository documents robot simulation and automation projects created in **KUKA.Sim**, the official offline programming and simulation software for KUKA industrial robots. Projects span both professional internship work at **OPmobility** (automotive plastic components manufacturer) and personal engineering explorations.
>
> KUKA.Sim allows engineers to model robotic workcells, simulate robot programs, optimize cycle times, and detect collisions before physical implementation — reducing commissioning risk and improving production efficiency.

> ## About KUKA.Sim
>
> KUKA.Sim is the industry-standard simulation platform for KUKA robots, enabling:
> - Virtual commissioning of robot programs
> - - Accurate 3D visualization of robotic workcells
>   - - Automatic code generation for KUKA controllers
>     - - Import of CAD models (STEP, IGES, etc.)
>       - - Cycle time measurement and optimization tools
